Economic Indicator Forecasting Dashboard
Business Context: The Economic Research department tracks numerous economic indicators (inflation, GDP, employment, etc.). Currently, forecasting models are run separately, and results are presented in static reports.
Challenge: Build an interactive AI-powered dashboard that:
•	Displays historical economic indicator trends
•	Provides short-term forecasts using ML models
•	Allows "what-if" scenario analysis
•	Explains forecast reasoning in plain language
•	Sends alerts for significant deviations
Technical Scope:
•	Time series forecasting (AWS Forecast or SageMaker)
•	Interactive visualization
•	Generative AI for explanations
•	Data pipeline from existing sources
Success Metrics:
•	Dashboard usability and insights
•	Clarity of AI explanations
Data:
•	Public FRED + BEA (US Bureau of Economic Analysis data
•	Link: U.S. Bureau of Economic Analysis (BEA) - https://bea.gov/
