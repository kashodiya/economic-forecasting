

## Environmental constraints
- This project is part of a hackathon event. The solution does not need to have scalability, performance, security etc. 
- Focus on creating a working solution that can show the value of the idea.
- This Windows PC is running on a AWS Workspace. This would be used as development machine.
- We have access to Bedrock LLMs.
- We have a EC2 Linux server. This would be our deployment machine.

## Technology choice
- Create Python app using FastAPI as backend.
- Frontend should be writtin in a single index.html.
- In the index.html file use VueJs, Vuetify and VueRouter from CDN.
- If you need database use local SQLite database.
- Keep it simple but effective.
- If you need to use LLM use AWS Bedrock model "us.anthropic.claude-sonnet-4-20250514-v1:0"
